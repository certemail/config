-check current partitions:
    # cat /proc/partitions      // check current partitions
    # lsblk                     // list block devices and nice summary
    # partprobe                 // push new changes to kernel, so re-read partition table

-find free disk space:
    # df -hT
    # findmnt
    # findmnt -D                // imitate output of df

-find unallocated disk space:
    # parted [DEVICE] print free

-creating file systems:
    # mkfs -t xfs /dev/vdb1     // add a label: use -L [LABEL_NAME]

-mounting filesystems:
    -use device block UUID (created and stored in superblock when fs is created
        # blkid [PARTITION]

    -check to ensure that it is entered properly in /etc/fstab:
        # mount -a

-creating and managing swap space:
    -create partition in fdisk/gdisk
    -set type of partition to "82 Linux Swap"
    -activate using: swapon [PARITION]

    # mkswap /dev/sda3          // applies swap signature to the partition
    # swapon /dev/sda3          // activate swap (turn on all: swapon -a)

    # swapoff                   // deactivate swap
    # swapon -s                 // get summary of swap
    # free -h                   // check usage of swap

    -to add swap to /etc/fstab:
    UUID=xxxxxxxxx-xxxxx-xxxxxx-xxxxx   swap    swap    defaults    0  0

    * UUID is stored in swap signature on the device (part of output of mkswap)
    * last 2 fields: dump and fsck order

    -swap priority can be added in /etc/fstab using mount option: pri=[PRIORITY_LEVEL]
    UUID=xxxxxxxxx-xxxxx-xxxxxx-xxxxx   swap    swap    pri=1    0  0

-repair filesystem:
    # umount /var
    # fsck -t xfs /dev/sda7
    # mount /dev/sda7 /var

-get filesystem information:
    # dumpe2fs /dev/sda4
    # dumpe2fs -h /dev/sda4 | grep features

-troubleshooting:
    # lsof [MOUNTPOINT]

-fdisk parition codes:
    82 -> swap
    83 -> Linux
    8e -> Linux LVM
